## Lernziel

Daten in Server Components laden und Client-Bundles klein halten.

## Kernideen

- Server Components koennen direkt `await` verwenden.
- Client Components nur fuer Interaktion einsetzen.
- Daten nah am Render-Ort laden.

## Beispiel

```tsx
export default async function UsersPage() {
  const users = await fetch("https://example.com/api/users").then((r) => r.json());
  return <pre>{JSON.stringify(users, null, 2)}</pre>;
}
```

## Warum React?

Mit React kannst du komplexe Oberflaechen aus kleinen wiederverwendbaren Bausteinen aufbauen.

## Komponenten

Eine Komponente ist eine Funktion, die UI zurueckgibt. Kombiniere kleine Komponenten statt grosse Templates zu schreiben.

## Props und State

- Props uebergeben Daten von Eltern- zu Kindkomponenten.
- State speichert Daten, die sich im Zeitverlauf aendern.

## Praxis

Erstelle eine `ProfileCard`-Komponente mit `name` und `role` als Props und rendere semantisches HTML.

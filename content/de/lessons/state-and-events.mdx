## Lernziel

Benutzereingaben korrekt in State-Updates umsetzen.

## Kernideen

- `useState` verwaltet lokale Werte.
- Event-Handler aktualisieren State ueber Setter.
- UI wird immer aus State abgeleitet.

## Beispiel

```tsx
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>{count}</p>
      <button onClick={() => setCount((v) => v + 1)}>Erhoehen</button>
    </div>
  );
}
```

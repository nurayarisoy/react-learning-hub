## Learning Goal

Control form values through React state and validate user input.

## Core Ideas

- Controlled inputs bind `value` + `onChange`.
- Keep form state close to form logic.
- Validate early and provide clear error messages.

## Example

```tsx
import { useState } from "react";

function EmailForm() {
  const [email, setEmail] = useState("");

  return (
    <form>
      <input value={email} onChange={(e) => setEmail(e.target.value)} />
    </form>
  );
}
```

## Learning Goal

Understand how JSX becomes elements and how React updates the screen efficiently.

## Core Ideas

- JSX is syntax sugar for `React.createElement` style calls.
- Components return JSX snapshots of UI.
- React compares previous and next trees to apply minimal DOM updates.

## Example

```tsx
function Welcome({ name }: { name: string }) {
  return <h1>Hello, {name}</h1>;
}
```

## Common Mistakes

- Returning multiple sibling elements without a wrapper or fragment.
- Mixing statements directly inside JSX blocks.

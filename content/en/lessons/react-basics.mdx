## Why React?

React helps you compose complex interfaces from small reusable pieces.

## Components

A component is a function that returns UI. Compose small components instead of writing large templates.

## Props and State

- Props pass data from parent to child.
- State stores data that changes over time.

## Practice

Create a simple `ProfileCard` component that receives `name` and `role` as props and renders semantic HTML.

## Learning Goal

Manage local UI state and react to user events.

## Core Ideas

- Use `useState` for local mutable values.
- Event handlers update state through setters.
- Derive UI from state, not manual DOM mutation.

## Example

```tsx
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>{count}</p>
      <button onClick={() => setCount((v) => v + 1)}>Increment</button>
    </div>
  );
}
```

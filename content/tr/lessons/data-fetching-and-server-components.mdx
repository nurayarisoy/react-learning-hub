## Hedef

Veriyi server componentlerde cekip client bundle boyutunu azaltmak.

## Temel Noktalar

- Server componentler dogrudan `await` kullanabilir.
- Client component sadece etkilesim gereken yerde kullanilir.
- Veri cekme kodu goruntulemeye yakin tutulur.

## Ornek

```tsx
export default async function UsersPage() {
  const users = await fetch("https://example.com/api/users").then((r) => r.json());
  return <pre>{JSON.stringify(users, null, 2)}</pre>;
}
```
